## Interactions

- Toggle **EN / 中文** at the top right
- Hover any node → highlight its connected path (industry ↔ capability ↔ technique)
- Click any **industry chip** (or industry node) → focus that industry's sub-tree
- Type in the **search box** (e.g. `XGBoost`, `AI 中台`, `Pricing`) → matched nodes light up
- Use `+` / `−` / `1:1` to zoom & reset
- Click any **timeline bar** at the bottom to see role highlights from my résumé
